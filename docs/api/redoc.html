Tecda Maniquí API (3.1.0)

Download OpenAPI specification:

Gestión integral de fábrica de maniquíes con arquitectura SOLID (TypeScript).

Cambios Recientes:

  • Validación de Stock Técnica: El ensamblaje ahora verifica la existencia de las 6 piezas (Cabeza, Torso, Brazos y Piernas).
  • Captura de Errores SQL: Errores del procedimiento almacenado se devuelven con descripción clara.
